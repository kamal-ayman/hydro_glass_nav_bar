# Screenshots Directory

Place your screenshot images here:

- `light_mode.png` - Screenshot of the tab bar in light mode
- `dark_mode.png` - Screenshot of the tab bar in dark mode
- `demo.gif` - Animated demo of the tab bar

These images are referenced in the pubspec.yaml and README.md files.

## Recommended Screenshot Sizes

- Static images: 300-400px width
- GIF demos: 300px width, optimized for file size

## Tips for Great Screenshots

1. Use a clean device frame or no frame at all
2. Show the tab bar with a beautiful background
3. Demonstrate the liquid glass effect clearly
4. Keep file sizes under 1MB for faster loading
